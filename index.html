Molnia Mobile

2023 - ...

Проект - приложение детского футбольного клуба, целью которого является повышение интереса маленьких футболистов к спорту с помощью карточный игры. За выполнение задач, поставленных тренером, игроки получают ценные игровые карточки.

Приложение состоит из трех глобальных составляющих - клиентского приложения игрока, тренера и администратора футбольного клуба. В зависимости от статуса пользователя, осуществившего вход, приложение предлагает различный интерфейс и функционал. Интерфейс приложения реализован на UIKit, с созданием многих сложных кастомных элементов, включая адаптивные ячейки таблиц и коллекций, кастомные вспылвающие окна и др. За бэкенд отвечает Firebase: для аутентификации и авторизации пользователей используется Auth, в качестве главной базы данных выступает Firestore, а файлы (пользовательские аватарки, картинки игровых карточек) хранятся в Cloud Storage.

Среди моих непосредственных задач были: верстка экранов, разработка сервисов взаимодействия с бэкендом и базой данных, разработка моделей данных для всего проекта.

Прогноз погоды

Репозиторий проекта на Github

Это курсовой проект курса Нетологии. Приложение для просмотра прогноза погоды. Интерфейс основан на Storyboard. Приложение построено с использованием паттерна MVVM. В данном приложение большую роль играет сервис работы с API, который использует фреймворк Alamofire. Приложение отправляет HTTP запросы к открытому API, полученные json данные декодирует в codable-модели. Локальное хранение данных организовано при помощи фреймворка CoreData. Также для реализации логики онбординга нового пользователя используется фреймворк UserDefaults. Кроме того, в данном приложении использованы фреймфорки для работы с геолокацией: CoreLocation, включая CLGeocoder. Приложение определяет местонахождения пользователя и предоставляет прогноз для этой местности. Также при поиске и добавлении места вручную используется прямой и обратный геокодинг при помощи CLGeocoder для перевода введенного названия места в координаты локации, необходимые для отправки запроса к API.

Для работы этого приложения использовался свободный API OpenMeteo.

Симулятор социальной сети

Репозиторий проекта на Github (WIP)

Это дипломный проект курса Нетологии. Приложение представлет собой клиент социальной сети. Полностью построен на архитектуре MVVM+C. Большое количество модулей сделало обоснованным внедрение координаторов для отимизации навигации и сохранении логики в коде.
